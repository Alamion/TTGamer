# Механика существ и НИП

Галактика кишит инопланетной дикой природой, от скромного курносого мопса до прожорливого раттара. Эта глава охватывает создание существ и НИП для вашей игры.

---

## Существа vs. Персонажи

Существа (неразумные создания) используют **упрощённую** версию листа персонажа:

| Аспект                  | Существо                                                                                            | Персонаж                        |
| ----------------------- | --------------------------------------------------------------------------------------------------- | ------------------------------- |
| **Физические Атрибуты** | Сила, Ловкость, Выносливость                                                                        | Те же                           |
| **Ментальные Атрибуты** | Восприятие, Интеллект, Хитроумие                                                                    | Те же                           |
| **Социальные Атрибуты** | —                                                                                                   | Харизма, Манипуляция, Внешность |
| **Способности**         | Только инстинктивные (Атлетика, Драка, Уклонение, Бдительность, Скрытность, Запугивание, Выживание) | Все 30                          |
| **Навыки Силы**         | —                                                                                                   | Полный набор                    |
| **Добродетели**         | —                                                                                                   | Совесть, Страсть, Самообладание |
| **Бюджет создания**     | Нет — существа не балансируются                                                                     | Очки Свободных Умений           |

Существа **не создаются по бюджету**. Ранкор просто сильнее эвока — здесь нет эквивалентности покупки очков. Правила конвертации ниже помогают перевести статистику существ WEG в эту систему.

---

## Масштаб существ

Существа используют ту же систему масштаба, что транспорт и персонажи (см. [Масштабы боя](../combat/combat-scales.mdx)). Большинство существ попадают в один из четырёх масштабов:

| Масштаб       | Примеры                                        | Как определить                               |
| ------------- | ---------------------------------------------- | -------------------------------------------- |
| **Вредитель** | Ковакская обезьяно-ящерица, майно́к, крыса-вумп | Меньше человека                              |
| **Персонаж**  | Человек, Вуки, большинство ПИ                  | Размером с человека                          |
| **Спидер**    | Таунтаун, дьюбек, банта                        | Достаточно велик, чтобы ездить верхом; 2–5 м |
| **Шагающий**  | Ранкор, раттар, вампа, AT-AT                   | Крупнее спидера; 5 м+                        |

### Определение масштаба по статистикам WEG

Если **Сила** существа WEG превышает **5D**, увеличьте масштаб на один шаг за каждую кость выше 5 и ограничьте Силу значением 5D. Ранкор (Сила 8D в WEG) получил бы +3 шага — Персонаж → Спидер → Шагающий → Капитальный — но ранкор явно не капитальный корабль; используйте здравый смысл. Масштаб «Шагающий» хорошо подходит ранкору.

Если у существа есть **размер** в описании, сравните его с размерами транспортных средств для грубого соответствия масштабу.

## Конвертация существ WEG

### Шаг 1: Вид, Тип, Масштаб, Размер

| Поле        | Источник                                                                       |
| ----------- | ------------------------------------------------------------------------------ |
| **Вид**     | Название существа из источника WEG.                                            |
| **Тип**     | Его экологическая роль (например, «Арктический хищник», «Болотный падальщик»). |
| **Масштаб** | Определяется по размеру и Силе (см. выше).                                     |
| **Размер**  | Физические размеры из описания (например, «3 м в высоту»).                     |

Опциональные поля для именованных существ:

| Поле         | Источник                                            |
| ------------ | --------------------------------------------------- |
| **Имя**      | Если существо — конкретная особь (питомец, скакун). |
| **Владелец** | Персонаж, которому оно принадлежит.                 |

### Шаг 2: Физические Атрибуты

**Сила.** Если показатель WEG 5D или меньше, используйте его напрямую (игнорируйте плюсы). Если выше 5D, установите на 5D и увеличьте масштаб соответственно (или назначьте преимущества вроде Мощный или Толстокожий для существ, которые сильны, но не намного крупнее).

**Ловкость.** То же, что и Сила — используйте напрямую, если 5D или меньше. Если выше 5D, ограничьте 5D и либо уменьшите масштаб до Вредителя, либо назначьте дополнительные кости на способность Уклонение, либо дайте преимущества вроде Кошачьих Рефлексов.

**Выносливость.** WEG использует Силу для сопротивления; эта система использует Выносливость. Дайте существу тот же рейтинг, что и его Сила. Для собственных существ без статистик WEG Сила и Выносливость не обязаны совпадать.

### Шаг 3: Ментальные Атрибуты

WEG даёт существам только один ментальный показатель: **Восприятие**. Используйте рекомендации ниже для определения всех трёх:

| Роль существа                                  | Восприятие | Интеллект | Хитроумие |
| ---------------------------------------------- | ---------- | --------- | --------- |
| **Преследуемая добыча** (травоядное, пугливое) | = WEG Вос  | −1D       | −1D       |
| **Одомашненное** (обучаемое, вьючное)          | −1D        | = WEG Вос | −1D       |
| **Хищник** (охотится, устраивает засады)       | −1D        | −1D       | = WEG Вос |
| **Минимальное сознание** (WEG Вос = 1D)        | 1D         | 1D        | 1D        |

Если Восприятие WEG выше 5D, ограничьте 5D и назначьте дополнительные кости на Бдительность или преимущества вроде Острый Слух или Шестое Чувство.

### Шаг 4: Сила Воли

Начните всех существ с базовой Силы Воли **5D**. Если в статистиках WEG указан показатель **Норов**, добавьте его (игнорируйте плюсы). Скорректируйте вверх или вниз на основе описания — особенно своенравный или покладистый вид может заслуживать собственного значения.

### Шаг 5: Способности

У существ есть только инстинктивные способности. Используйте эти конвертации:

| Способность существа                                 | Источник WEG                                                                                                                                                            |
| ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Атлетика**                                         | Показатель Move ÷ 5, округлённый вниз. Охватывает бег, полёт, плавание, рытьё.                                                                                          |
| **Драка**                                            | 1D за тип атаки (коготь, укус, хвост и т.д.).                                                                                                                           |
| **Бдительность, Скрытность, Запугивание, Выживание** | Соответствует любым навыкам, указанным в блоке статистики WEG. Если блок статистики WEG даёт бонус навыка, вычтите родительский атрибут, чтобы найти кости способности. |

Ограничьте все способности **5D**. Если значение превышает 5 костей и масштаб существа не компенсирует это, назначьте соответствующее Преимущество.

### Шаг 6: Бой

**Броня.** У некоторых существ есть твёрдая кожа или чешуйчатые пластины. Запишите показатель Брони WEG напрямую, с ограничением до 5.

**Атаки.** Для каждого типа атаки (коготь, укус, удушение и т.д.) укажите название и урон. Урон почти всегда основан на Силе. Скопируйте урон WEG, игнорируя плюсы после костей.

| Запись WEG         | Конвертировано        |
| ------------------ | --------------------- |
| `Коготь: СИЛ+2D+2` | **Коготь** (Л) СИЛ+2D |
| `Укус: СИЛ+1D`     | **Укус** (Л) СИЛ+1D   |
| `Таран: СИЛ+2D`    | **Таран** (Д) СИЛ+2D  |

Используйте здравый смысл для типа урона:

- Когти, зубы, жала → **Летальный**
- Таран, удар, хвост → **Дробящий**

### Шаг 7: Преимущества и Недостатки

Перенесите соответствующие Преимущества и Недостатки из описания WEG. Распространённые преимущества существ:

- **Мощный** (+X костей к броскам, связанным с Силой)
- **Толстокожий** (+X костей к сопротивлению)
- **Адаптация к среде** (холод, жара, вода и т.д.)
- **Дополнительная конечность** (хвост, щупальца — считается как дополнительные конечности)
- **Природное Оружие** (когти, клыки — существующая атака, но формализует черту)
- **Невербальный язык** (вой, щебет — ограниченное общение)
- **Камуфляж** (бонус к Скрытности в определённой среде)

### Шаг 8: Здоровье

Существа используют стандартный трек здоровья на 7 уровней. Для стай или стад отслеживайте несколько идентичных существ на одном листе — нет необходимости в отдельном листе на каждую особь.

Существо в состоянии Выведен из строя может быть обойдено или добито по усмотрению Рассказчика; большинство хищников убегает, получив ранение (Изуродован или хуже). Загнанные в угол существа сражаются отчаянно — Рассказчик может не применять штраф за ранение к последнему бою существа.

## НИП

Неигровые персонажи используют **полный лист персонажа** с одним различием: являются ли они **Именованными** или **Неименованными**.

### Именованные НИП

Именованные НИП создаются точно как игровые персонажи — те же распределения атрибутов/способностей/добродетелей, очки предысторий и очки свободных умений. Они:

- Используют **полный лист персонажа**
- **Могут сопротивляться Летальному урону** (героическое правило действует — минимум 1 рана)
- Имеют **имя и личность**
- Развиваются через Опыт (по усмотрению Рассказчика)

Используйте правила создания персонажа из Фазы 2 для создания именованных НИП. Для быстрой генерации назначьте:

- **Атрибуты:** Первичные 4, Вторичные 3, Третичные 2 (или 3/2/1 для второстепенных именованных НИП)
- **Способности:** 13/9/5 как обычно, с предпочтением навыков, соответствующих их роли
- **Предыстории:** 5 очков
- **Добродетели:** 7 очков
- **Очки Свободных Умений:** 15

### Неименованные НИП (Мобы)

Неименованные НИП — штурмовики, пираты, барные вышибалы — упрощены:

- **Атрибуты:** 2 или 3 в соответствующих показателях, 1 во всём остальном
- **Способности:** 2 или 3 в навыках, соответствующих работе, 0 во всём остальном
- **Добродетели:** Совесть 2, Страсть 2, Самообладание 2
- **Сила Воли:** 4
- **Нет очков свободных умений**
- **Не могут сопротивляться Летальному урону** — любое попадание наносит полный урон
- **Один уровень здоровья = выведен из строя** (опционально: трек здоровья на 3 блока вместо 7)

#### Пример блока статистики моба: Штурмовик

| Атрибут      | Кости | Способность    | Кости |
| ------------ | ----- | -------------- | ----- |
| Сила         | 3     | Бластеры       | 3     |
| Ловкость     | 3     | Уклонение      | 2     |
| Выносливость | 3     | Рукопашный бой | 2     |
| Восприятие   | 2     | Драка          | 2     |
| Интеллект    | 2     | Атлетика       | 2     |
| Хитроумие    | 2     | Бдительность   | 2     |

Броня: Штурмовик (АР +3D, Лов −2D). Оружие: Бластерная винтовка (9D ур, дальность 200, патронов 50). Сила Воли: 4. Сопротивление: Выносливость 3 + броня 3D = 6D.

### Быстрые броски НИП

Для второстепенных НИП, где вам не нужна полная статистика, назначьте **пул костей** для их специализации:

| Важность НИП                                                            | Пул специализации | Вторичный пул |
| ----------------------------------------------------------------------- | ----------------- | ------------- |
| **Моб** (бармен, случайный охранник)                                    | 4D                | 2D            |
| **Компетентный** (ветеран-штурмовик, охотник за головами)               | 6D                | 4D            |
| **Элита** (Имперский королевский гвардеец, лидер охотников за головами) | 8D                | 6D            |

Совершите бросок пула специализации со сложностью 6 для любой задачи в рамках их области знаний. Используйте вторичный пул для всего остального.

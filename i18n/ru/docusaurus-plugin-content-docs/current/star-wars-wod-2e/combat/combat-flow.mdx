# Ход боя

Когда бластеры начинают стрелять, а световые мечи зажигаться, игра переходит в структурированные раунды. Вот как разворачивается бой.

---

## 1. Инициатива

В начале боя каждый персонаж совершает бросок **Хитроумия + Бдительность** со сложностью **6**. Персонажи действуют в порядке убывания успехов. Связки разрешаются одновременно — оба объявляют действия и выполняют их вместе.

<TWWrapper>
    <InlineRoll notation={'4d10>=6f=1'} variant="details" preroll /> — бросок инициативы для
    Хитроумие 3 + Бдительность 1
</TWWrapper>

Инициатива определяется **один раз** в начале боя, а не каждый ход. Порядок остаётся фиксированным на всю сцену, если только персонаж намеренно не выжидает или не задерживает своё действие.

:::note[Внезапность и Засада]
Если одна сторона застаёт другую врасплох, атакующие получают **бесплатный ход** до броска инициативы. После раунда внезапности инициатива бросается обычным образом.
:::

---

## 2. Структура хода

Каждый ход персонаж может совершить **одно перемещение** и **одно действие**.

### Перемещение

Переместиться на разумное расстояние — нырнуть за укрытие, сблизиться для рукопашной, сменить позицию для лучшего угла. В тактическом выражении это примерно **3–5 метров за ход** для большинства персонажей. Быстрые расы или открытая местность могут позволять больше.

### Действия

Основное действие покрывает всё, что требует броска:

| Тип действия             | Примеры                                                           |
| ------------------------ | ----------------------------------------------------------------- |
| **Атака**                | Выстрелить из бластера, ударить виброклинком, пустить кулак в ход |
| **Активная защита**      | Объявить реакцию Уклонения                                        |
| **Использование навыка** | Починить панель, взломать терминал, оказать медицину              |
| **Активация Силы**       | Сосредоточиться на способности Силы                               |
| **Разное**               | Достать оружие, нажать рубильник, выкрикнуть приказ               |

### Разделение пула

Чтобы сделать больше одного действия за ход, разделите наименьший соответствующий пул поровну. Каждое действие получает свой бросок. Перемещение стоит **1 кость за 3 ярда** из выбранного вами пула.

<TWWrapper>
    <InlineRoll notation={'5d10>=6f=1'} variant="details" preroll /> — пример разделённого пула
</TWWrapper>

См. [Пулы костей — Разделение пулов](../core-rules/dice-pools.mdx#разделение-запасов-на-несколько-действий) для полных правил.

## 3. Разрешение дальнего боя

### Шаг 1: Объявление и бросок

Совершите бросок **Ловкости + Бластеры** со сложностью **6**. Каждый успех — потенциальное попадание.

<TWWrapper>
    <CharRoll attr="Dexterity" skill="Blaster" variant="details" />
</TWWrapper>

### Шаг 2: Активная защита (Уклонение)

Если цель ещё не действовала в этот ход, она может объявить реакцию **Уклонения** — бросок **Ловкости + Уклонение** со сложностью **6**. Каждый успех на броске Уклонения вычитает **один успех** из броска атакующего, как по Правилу Единицы.

<TWWrapper>
    <CharRoll attr="Dexterity" skill="Dodge" variant="details" />
</TWWrapper>

Если цель уже действовала в этот ход, она не может объявить Уклонение, но всё ещё может использовать **пассивную защиту** — Мастер может повысить сложность атаки в зависимости от укрытия, дистанции или движения (обычно +1 до +3).

Как только персонаж объявляет Уклонение, он не может совершать никаких других действий в этот ход — всё его внимание сосредоточено на избегании атаки.

### Шаг 3: Определение попадания

Если у атакующего остался **хотя бы 1 успех** после вычитания Уклонения, атака попадает. Дополнительные успехи сверх первого не увеличивают урон напрямую (если только не используются прицельные выстрелы или специальные боеприпасы) — они просто означают более чистое попадание.

### Шаг 4: Бросок урона

Совершите бросок костей урона оружия со сложностью **6**. Каждый успех наносит **один уровень здоровья** урона.

<TWWrapper>
    <InlineRoll notation={'5d10>=6'} variant="details" preroll /> — стандартный бластерный пистолет
    (5D урона)
</TWWrapper>

### Шаг 5: Сопротивление (Soak)

Цель совершает бросок **Выносливости** со сложностью **6**. Каждый успех отменяет один уровень здоровья урона.

<TWWrapper>
    <CharRoll attr="Stamina" variant="details" />
</TWWrapper>

:::caution[Барьер Героического Правила]
ПИ и именованные НИП могут сопротивляться Летальному урону, но они **всегда получают минимум 1 уровень ран** независимо от успехов сопротивления. Неименованные НИП (мобы) **вообще не могут сопротивляться Летальному урону** — они получают полный урон от каждого попадания.
:::

### Шаг 6: Применение урона

Отметьте урон на треке здоровья. Примените штрафы за ранения к будущим броскам.

| Тип урона      | Отметка | Может сопротивляться?       | Мин. рана |
| -------------- | ------- | --------------------------- | --------- |
| **Дробящий**   | `/`     | Все                         | Нет       |
| **Летальный**  | `X`     | Только ПИ + именованные НИП | **1**     |
| **Оглушающий** | `•`     | Все (как Дробящий)          | Нет       |

## 4. Разрешение рукопашного боя

Рукопашный бой использует ту же структуру, что и дальний, с двумя отличиями:

1. **Пул:** Бросок **Ловкости + Рукопашный бой** (или **Силы + Драка** для безоружной).
2. **Нет штрафа к Уклонению:** Персонаж, объявивший Уклонение от рукопашной, всё ещё теряет своё действие — но также может попытаться парировать (см. [Дуэльные правила](#6-дуэльные-правила-опционально)) вместо уклонения.

Если цель находится на расстоянии вытянутой руки, дистанция не учитывается. Сближение для рукопашной требует перемещения.

<TWWrapper>
    <CharRoll attr="Dexterity" skill="Melee" variant="details" />
</TWWrapper>

### Безоружный бой

Драка использует **Силу + Драка** и наносит **Дробящий** урон, равный **Сила + 1**. Персонажи с преимуществом **Природное Оружие** наносят Летальный урон вместо Дробящего.

## 5. Реакции

У персонажа есть одна **реакция** за ход — рефлекторный ответ на непосредственную угрозу.

| Реакция         | Триггер                                        | Эффект                                                                    |
| --------------- | ---------------------------------------------- | ------------------------------------------------------------------------- |
| **Уклонение**   | Атака нацелена на вас (дальняя или рукопашная) | Бросок Ловкости + Уклонение; успехи вычитаются из успехов атакующего      |
| **Парирование** | Атака нацелена на вас (только рукопашная)      | Бросок Ловкости + Рукопашный бой; успехи вычитаются из успехов атакующего |
| **Перехват**    | Противник проходит мимо вас                    | Бросок, чтобы остановить; считается за действие в следующий ход           |

Объявление реакции использует ваше действие на ход. Вы не можете Уклониться или Парировать, если уже действовали.

---

## 6. Дуэльные правила (опционально)

Для кульминационных поединков один на один — дуэли на световых мечах, разборки на виброклинках, драматичные кулачные бои — стандартный подход «бросай, пока кто-то не попадёт» может казаться плоским. Для таких сцен используйте фреймворк **Расширенного + Сопротивляемого** действия ниже.

Для большинства рукопашных атак стандартных правил боя достаточно.

### Подготовка

1. Совершите бросок **Инициативы** (Хитроумие + Бдительность). Победитель становится **Атакующим**; проигравший — **Защищающимся**.
2. Эти роли сохраняются, пока Защищающийся не перехватит инициативу.
3. Оба дуэлянта начинают **Расширенное действие**, накапливая успехи ход за ходом к своей цели.

### Цель

| Роль             | Требуется                                   | Результат при достижении цели                                                            |
| ---------------- | ------------------------------------------- | ---------------------------------------------------------------------------------------- |
| **Атакующий**    | Успехов, равных **Силе Воли Защищающегося** | Совершить стандартную рукопашную атаку — урон, сопротивление и штрафы применяются обычно |
| **Защищающийся** | Успехов, равных **Силе Воли Атакующего**    | Перехватить инициативу — роли меняются, успехи сбрасываются                              |

### Темп и длительность

Типичная дуэль длится **3–5 ходов за фазу** — сопоставимо с несколькими раундами стандартного боя.

- Если оба дуэлянта примерно равны (похожая Сила Воли, похожий Рукопашный бой), ожидайте ~3 хода для достижения цели при ~2 успехах за ход.
- Умные дуэлянты используют **слабейшую группу Атрибутов** противника. Если вы ведёте дуэль, используя Атрибут из **третичной** группы противника (той, на которую он потратил меньше всего очков), уменьшите цель на 3. Если вы используете его **первичную** группу — увеличьте на 3. Это поощряет творческие комбинации Атрибут+Способность:
    - **Восприятие + Рукопашный бой** для изучения паттернов (Ментальное против грубой силы)
    - **Сила + Рукопашный бой** чтобы подавить (Физическое против учёного)
    - **Хитроумие + Рукопашный бой** чтобы перехитрить (Ментальное против драчуна)

### Бросок

Каждый ход оба бойца совершают бросок соответствующего Атрибута + Рукопашный бой. Рассказчик поощряет творческие комбинации — **Восприятие + Рукопашный бой** для изучения паттернов противника, **Сила + Рукопашный бой** чтобы подавить, **Хитроумие + Рукопашный бой** чтобы перехитрить.

<TWWrapper>
    <CharRoll attr="Dexterity" skill="Melee" variant="details" />
</TWWrapper>

### Разрешение дуэли

Когда **Атакующий** достигает своей цели, он немедленно совершает стандартную рукопашную атаку — при попадании переходит к урону и сопротивлению как обычно. Атакующий может продолжать совершать стандартные атаки в каждый последующий ход, пока сохраняет инициативу.

Когда **Защищающийся** достигает своей цели, он перехватывает инициативу. Роли меняются. Накопленные успехи сбрасываются до нуля. Бой продолжается с новым Атакующим, развивающим своё преимущество.

### Нарушение ритма

Любой дуэлянт может в свой ход отказаться от броска Рукопашного боя и совершить другое действие — пнуть предмет мебели, бросить песок, активировать оборудование. Это аннулирует прогресс в расширенном действии, но может создать тактические возможности.

## 7. Укрытие и позиционирование

| Ситуация                                         | Эффект                                            |
| ------------------------------------------------ | ------------------------------------------------- |
| **Лёгкое укрытие** (ящик, стол)                  | +1 к сложности попадания                          |
| **Тяжёлое укрытие** (стена, бронированная дверь) | +2 к сложности попадания                          |
| **Лёжа / пригнувшись**                           | +1 к сложности попадания; −1 кость к своим атакам |
| **Цель бежит**                                   | +1 к сложности попадания                          |
| **В упор** (в пределах 2 м)                      | −1 к сложности попадания                          |

Укрытие — это разница между жизнью и смертью в перестрелке. Умное позиционирование сохранит вам жизнь.

Для полного разбора вероятностей успеха, частоты провалов и калибровки сложности для вашего стола см. **[Справочник сложности и успехов](/docs/star-wars-wod-2e/gm/difficulty-reference)**.

## 8. Штрафы за ранения

По мере накопления урона каждое действие страдает:

| Уровень здоровья | Штраф                |
| ---------------- | -------------------- |
| Синяк            | —                    |
| Потрёпан         | **-1**               |
| Ранен            | **-2**               |
| Тяжело ранен     | **-3**               |
| Изуродован       | **-4**               |
| Искалечен        | **-5**               |
| Выведен из строя | Не может действовать |

Эти кости вычитаются напрямую из каждого пула костей. Персонаж в состоянии Тяжело ранен (-3) с Ловкостью 5 + Бластеры 3 = 8 костей бросает всего **5 костей**.

## Сводка: Схема раунда боя

```
1. Бросок Инициативы (Хитроумие + Бдительность)
         ↓
2. Наибольший успех действует первым
         ↓
3. В ваш ход:
   ├─ Перемещение (3-5 м, или разделить пул для большего)
   └─ Действие:
       ├─ Атака → Бросок пула → Цель может Уклониться → Урон → Сопротивление
       ├─ Использовать навык → Бросок как обычно
       ├─ Активировать Силу → Следовать правилам Силы
       └─ Выжидать → Действовать позже в порядке хода
         ↓
4. Повторять, пока бой не закончится
```

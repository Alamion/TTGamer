# Масштабы боя

Галактика полна вещей, которые гораздо больше — или гораздо меньше — человека. Ранкор, бьющий эвока, и X-wing, штурмующий Звезду Смерти, используют одну и ту же механику: **масштабы боя**.

---

## Таблица масштабов

Найдите масштаб атакующего и масштаб цели в этой таблице. Сосчитайте строки между ними — каждая это один **шаг**.

<TWWrapper>
    <ScaleChart />
</TWWrapper>

- **Одинаковый масштаб** — без модификаторов. Обычные правила боя.
- **Атакующий крупнее** (выше в таблице) — сложнее попасть (+1 к Сл. за шаг), легче нанести урон (−1 к Сл. за шаг).
- **Атакующий мельче** (ниже в таблице) — легче попасть (−1 к Сл. за шаг), сложнее нанести урон (+1 к Сл. за шаг).

---

## Применение модификатора

Сосчитайте шаги между двумя масштабами. За каждый шаг:

| Направление             | Попадание                            | Урон                                      |
| ----------------------- | ------------------------------------ | ----------------------------------------- |
| Крупный атакует мелкого | **+1 к сложности** (сложнее попасть) | **−1 к сложности** (легче нанести урон)   |
| Мелкий атакует крупного | **−1 к сложности** (легче попасть)   | **+1 к сложности** (сложнее нанести урон) |

### Крупный против Мелкого — Атака сверху

Крупному атакующему трудно попасть в маленькую, юркую цель, но если он попадает, урон сокрушителен.

<TWWrapper>
    <InlineRoll notation={'6d10>=8'} variant="details" preroll />
</TWWrapper>

Ранкор (Шагающий), атакующий эвока (Персонаж) — 2 шага вниз. Сложность попадания **6 + 2 = 8**. При попадании сложность урона **6 − 2 = 4**.

### Мелкий против Крупного — Атака снизу

Мелкому атакующему легче попасть по огромной цели, но его оружию трудно нанести значимый урон.

<TWWrapper>
    <InlineRoll notation={'8d10>=4'} variant="details" preroll />
</TWWrapper>

Эвок, стреляющий из бластера в ранкора (Персонаж против Шагающего, 2 шага вверх). Сложность попадания **6 − 2 = 4**. Сложность урона **6 + 2 = 8**.

## Примеры

### Ранкор против Эвока (Рукопашная)

| Фактор    | Значение                                                          |
| --------- | ----------------------------------------------------------------- |
| Масштабы  | Шагающий (Ранкор) → Персонаж (Эвок)                               |
| Шаги      | **2**                                                             |
| Попадание | 6 + 2 = **8** (Ранкору трудно прихлопнуть юркого эвока)           |
| Урон      | 6 − 2 = **4** (если попадает, массивные когти сокрушают насквозь) |

### X-wing против Звезды Смерти (Космический бой)

| Фактор    | Значение                                            |
| --------- | --------------------------------------------------- |
| Масштабы  | Звездолёт (X-wing) → Капитальный (Звезда Смерти)    |
| Шаги      | **2**                                               |
| Попадание | 6 − 2 = **4** (Звезда Смерти — огромная цель)       |
| Урон      | 6 + 2 = **8** (лазерные пушки едва царапают корпус) |

X-wing понадобятся протонные торпеды (высокий урон) или прицельный залп по уязвимой точке (генератор щита, мостик), чтобы добиться реального эффекта.

### A-wing против TIE Истребителя (Одинаковый масштаб)

| Фактор    | Значение              |
| --------- | --------------------- |
| Масштабы  | Звездолёт → Звездолёт |
| Шаги      | **0**                 |
| Попадание | стандартные **6**     |
| Урон      | стандартные **6**     |

Честный бой. Мастерство пилота определяет исход.

## Определение масштаба

Не уверены, какой у чего масштаб? Используйте эти рекомендации:

| Масштаб           | Примеры                             | Как определить                                               |
| ----------------- | ----------------------------------- | ------------------------------------------------------------ |
| **Звезда Смерти** | Звезда Смерти, Старкиллер           | Может пристыковать Капитальный корабль                       |
| **Капитальный**   | Супер-Звезда Смерти, Home One       | Экипаж 10+, может принимать звездолёты                       |
| **Транспорт**     | Тысячелетний Сокол, Призрак         | Экипаж < 10, перевозит пассажиров/груз, выходит в космос     |
| **Звездолёт**     | X-wing, TIE Истребитель, A-wing     | Выходит в космос, без пассажиров, < 1 тонны груза            |
| **Шагающий**      | AT-AT, AT-ST, Ранкор                | Наземное транспортное средство на ногах/гусеницах, не летает |
| **Спидер**        | Лэндо-спидер, спидер-байк, MTT      | Летает или парит, не может выйти в космос                    |
| **Персонаж**      | Человек, Вуки, Эвок, большинство ПИ | Существо размером с человека                                 |
| **Вредитель**     | Ковакская обезьяно-ящерица, майно́к  | Меньше человека, размером с вредителя                        |

Существа крупнее человека, но меньше шагающего (например, ранкор или раттар) относятся к масштабу **Шагающий**. Существа меньше человека — к масштабу **Вредитель**.

---

## Транспортные средства и масштабы боя

Бой с участием транспортных средств использует ту же систему масштабов. Звездолёт, атакующий капитальный корабль, следует правилам «атаки снизу» — легче попасть, сложнее нанести урон.

Бой на транспортных средствах добавляет **Манёвренность** (бонус костей к броскам пилота) и **Прочность** (используется для сопротивления вместо Выносливости). Бросок сопротивления транспортного средства использует его рейтинг Прочности со сложностью 6, модифицированный масштабом.

# Награды и развитие

Персонажи растут через опыт. В этой главе рассказывается, как начислять Опыт, сколько стоят улучшения и что Рассказчик должен требовать перед тратой очков.

---

## Начисление Опыта

В конце каждой сессии начисляйте **1–5 Опыта** на персонажа. В конце сюжетной арки начисляйте **1–3 бонусных Опыта**.

| Критерий          | Опыт | Когда начислять                                                                                                  |
| ----------------- | ---- | ---------------------------------------------------------------------------------------------------------------- |
| **Автоматически** | 1    | Каждая сессия. Просто за то, что пришли.                                                                         |
| **Обучение**      | 1    | Персонаж узнал что-то новое. Попросите игрока описать, что.                                                      |
| **Игра**          | 1    | Игрок хорошо отыграл роль — занимательно и уместно. Награждайте лучшего отыгрывающего сессии.                    |
| **Натура/Маска**  | 1    | Игрок убедительно отыграл Натуру и Маску своего персонажа.                                                       |
| **Героизм**       | 1    | Персонаж рисковал собой ради других. Не позволяйте игрокам фармить это — есть грань между героизмом и глупостью. |
| **Успех**         | 1    | Группа достигла своей цели (или сделала значимый прогресс).                                                      |
| **Опасность**     | 1    | Персонаж пережил значительную опасность.                                                                         |
| **Мудрость**      | 1    | Идея или понимание игрока изменили ход событий.                                                                  |

### Рекомендации по темпу

| Темп                                | Опыта за сессию | Показатели улучшаются до... |
| ----------------------------------- | --------------- | --------------------------- |
| **Медленный** (хроника на год)      | 2–3             | 4–5 костей после 20 сессий  |
| **Умеренный** (арка на 6 месяцев)   | 4–5             | 5–6 костей после 20 сессий  |
| **Быстрый** (трилогия в стиле кино) | 6–8             | 7+ костей после 20 сессий   |

Показатель может быть повышен **только на один пункт за сюжет**. Нельзя купить с 2 до 5 за один раз.

## Трата Опыта

### Стандартная стоимость показателей

| Показатель                            | Стоимость (Опыт)      |
| ------------------------------------- | --------------------- |
| **Атрибут**                           | текущий рейтинг × 4   |
| **Способность** (Талант/Навык/Знание) | текущий рейтинг × 2   |
| **Новая Способность**                 | 3                     |
| **Добродетель**                       | текущий рейтинг × 2   |
| **Сила Воли**                         | текущий рейтинг       |
| **Предыстория**                       | текущий рейтинг × 2   |
| **Навык Силы**                        | текущий рейтинг × 2   |
| **Новая Сила Силы**                   | 2                     |
| **Новый Навык Силы**                  | 3 (требуется учитель) |

_Пример: Повышение Способности с 2 до 3 стоит 2 × 2 = 4 Опыта. Повышение Атрибута с 3 до 4 стоит 3 × 4 = 12 Опыта._

### Сопротивление Тёмной стороне

СТС использует **обратную стоимость**: чем ниже ваш текущий рейтинг, тем дороже его повысить.

| Текущая СТС | Стоимость повышения на 1 |
| ----------- | ------------------------ |
| 0           | 10 Опыта                 |
| 1           | 9 Опыта                  |
| 2           | 8 Опыта                  |
| 3           | 7 Опыта                  |
| 4           | 6 Опыта                  |
| 5           | 5 Опыта                  |
| 6           | 4 Опыта                  |
| 7           | 3 Опыта                  |
| 8           | 2 Опыта                  |
| 9           | 1 Опыта                  |

### Навыки Силы и Силы

Когда пользователь Силы тратит Опыт на **повышение Навыка Силы**, он также **изучает одну новую Силу**, которая зависит от этого навыка — без дополнительной платы. Если он хочет конкретную Силу, связанную с навыком, который он не повышает, он платит **2 Опыта** за её отдельное изучение.

## Тренировка и сюжетные требования

Одного Опыта недостаточно. Рассказчик должен требовать:

- **Атрибуты** — Персонаж активно использовал Атрибут и расширял его пределы. Рост Силы требует физического напряжения; рост Интеллекта требует учёбы или решения проблем.
- **Способности** — Персонаж практиковал или использовал Способность в значимых ситуациях. Новая Способность требует обучения (учитель, руководство или метод проб и ошибок).
- **Добродетели** — Персонаж прошёл моральное испытание. Добродетели не могут быть повышены в вакууме.
- **Сила Воли** — Должно было произойти что-то, что сделало персонажа более уверенным — выживание в испытании, преодоление страха, неподчинение авторитету.
- **Навыки Силы** — Учитель (голокрон, Магистр-джедай, дух ситха) или интенсивное личное открытие. Сила не поддаётся поверхностному изучению.
- **Предыстории** — Трата Опыта на Предысторию означает, что персонаж заслужил её в игре — новый союзник, приобретённая репутация, полученный ресурс.
- **Человечность/СТС** — См. [Сопротивление Тёмной стороне](../character/dark_side_resistance.mdx).

Чем больше вы заставляете игроков оправдывать своё развитие через сюжет, тем более захватывающей становится игра.

## Очки Свободных Умений (Создание персонажа)

Создание персонажа использует **Очки Свободных Умений** вместо Опыта. Стоимость отличается.

| Показатель                       | Стоимость Свободных Умений       |
| -------------------------------- | -------------------------------- |
| **Атрибут**                      | 5 за пункт                       |
| **Способность**                  | 2 за пункт                       |
| **Добродетель**                  | 2 за пункт                       |
| **Предыстория**                  | 1 за пункт                       |
| **Навык Силы**                   | 3 за пункт                       |
| **Сила Силы**                    | 2 за силу                        |
| **Сопротивление Тёмной стороне** | 1 за пункт                       |
| **Сила Воли**                    | 2 за пункт                       |
| **Преимущество**                 | зависит (см. Руководство игрока) |

Стартовые Очки Свободных Умений: **15** (люди), корректируется расой. См. [Расы](../character/species.mdx) для нечеловеческих корректировок.

### Сравнение Свободных Умений и Опыта

| Показатель        | Стоимость Свободных Умений | Стоимость Опыта |
| ----------------- | -------------------------- | --------------- |
| Пункт Атрибута    | 5                          | текущий × 4     |
| Пункт Способности | 2                          | текущий × 2     |
| Пункт Добродетели | 2                          | текущий × 2     |
| Пункт Силы Воли   | 2                          | текущий         |
| Пункт Предыстории | 1                          | текущий × 2     |
| Пункт Навыка Силы | 3                          | текущий × 2     |
| Сила Силы         | 2                          | 2               |

## Развитие дроидов

Дроиды не тратят Опыт на Атрибуты или Способности так же, как живые персонажи:

- **Атрибуты** улучшаются как системы транспорта. Они требуют кредитов, запчастей и броска Техники (не Опыта). См. [Модификации](../vehicles-mechanisms/modifications.mdx).
- **Способности (программирование)** могут быть улучшены Опытом обычным образом или перепрограммированы во время простоя с помощью броска Техники (Компьютеры).
- **Новые Способности** требуют поиска совместимого программирования — купленного, скопированного с другого дроида или написанного с нуля.

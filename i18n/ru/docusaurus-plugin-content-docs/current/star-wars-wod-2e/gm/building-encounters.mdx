# Создание столкновений

Каждое столкновение — это препятствие между персонажами и их целью. В этой главе рассказывается, как создавать столкновения — от перестрелок с мобами до напряжённых социальных переговоров — используя инструменты как системы Рассказчика, так и этой конвертации.

---

## Уровни НИП

Не все НИП созданы равными. Система использует три уровня, чтобы бой оставался быстрым и драматичным.

### Именованные НИП (Антагонисты)

Именованные НИП используют **полные правила создания персонажа**. У них есть Атрибуты, Способности, Добродетели, Сила Воли, Очки Силы и полный трек здоровья на 7 уровней. Они могут сопротивляться Летальному урону, как и ПИ.

Используйте именованных НИП для:

- **Повторяющихся злодеев** (соперник-охотник за головами, имперский офицер, тёмный джедай)
- **Босс-столкновений** (криминальный авторитет хатт, охотник на ведьм, аколит ситхов)
- **Союзников и соперников**, важных для сюжета

### Неименованные НИП (Мобы)

Неименованные НИП используют упрощённый блок статистики — несколько ключевых показателей и общий пул здоровья. Они **не могут сопротивляться Летальному урону**. Каждое успешное попадание снижает их уровень здоровья. Это сохраняет бои быстрыми и позволяет героям чувствовать себя могущественными.

| Показатель                   | Обычный моб                           | Крепкий моб                     |
| ---------------------------- | ------------------------------------- | ------------------------------- |
| **Пулы костей**              | 3–5D                                  | 5–7D                            |
| **Уровни здоровья**          | 3 (Потрёпан, Ранен, Выведен из строя) | 5 (полный трек до Тяжело ранен) |
| **Сопротивление Летальному** | Нет                                   | Нет                             |
| **Сила Воли**                | 3                                     | 4                               |

Используйте мобов для:

- **Штурмовиков** (на удивление неточны, но опасны в численности)
- **Уличных бандитов, пиратов, наёмников**
- **Пилотов TIE истребителей** — незначительный транспорт тоже не получает броски сопротивления

### Существа

Существа следуют правилам главы [Механика существ](../creatures/mechanics.mdx). У них упрощённые показатели с модификаторами масштаба и особыми способностями. Некоторые — мобы (крыса, майно́к), некоторые — угрозы уровня именованных (ранкор, крайт-дракон).

## Типы действий

Каждое столкновение разрешается через броски костей. Рассказчик выбирает, какой тип действия подходит к ситуации.

### Простые действия

**Простое действие** требует только одного успеха. Большинство боевых атак, рутинные проверки навыков и быстрые задачи используют простые действия.

</TWWrapper>

_Один бросок. Одного успеха достаточно._

**Автоматический успех:** Если пул костей равен или превышает сложность, персонаж преуспевает автоматически без броска. Это для действий, которые персонаж может выполнить с закрытыми глазами.

Игрок может потратить **очко Силы Воли**, чтобы получить один автоматический успех на любом одиночном броске.

### Расширенные действия

**Расширенное действие** требует накопления нескольких успехов за несколько бросков. Используйте расширенные действия для задач, где временное давление или риск провала создают драму.

| Задача                                             | Требуемые успехи |
| -------------------------------------------------- | ---------------- |
| Забраться на дерево                                | 7                |
| Ремонт Лёгкого повреждения транспорта (постоянный) | 5 × масштаб      |
| Перепрограммирование дроида                        | 3–5              |
| Исследование архива данных                         | 5–15             |

Персонаж совершает бросок каждый ход (или час, для задач в простое), пока не накопит требуемое количество успехов. **Провал** сбрасывает прогресс до нуля, и Рассказчик может решить, что задача не может быть повторена.

### Сопротивляемые действия

В **сопротивляемом действии** два персонажа бросают друг против друга. Тот, у кого больше успехов, побеждает. Успехи противника отменяют избыток победителя — так что выдающиеся успехи редки.

**Примеры:** Слежка за целью, армрестлинг, дебаты, погоня.

### Расширенные и Сопротивляемые действия

Оба персонажа накапливают успехи с течением времени. Первый, достигший целевого числа, побеждает. Каждый ход только чистые успехи (ваши минус их) засчитываются в общий счёт.

**Пример:** Кинематографичная дуэль, хакерская гонка, преследование по криминальному дну Корусанта.

## Установка сложности

### Стандартная сложность

Базовая сложность для большинства бросков — **6**. См. **[Справочник сложности и успехов](./difficulty-reference.mdx)** для полной лестницы (Сл. 3–10) и интерактивного калькулятора вероятностей.

### Модификация сложности

Применяйте модификаторы в зависимости от ситуации:

| Обстоятельство                                  | Модификатор                |
| ----------------------------------------------- | -------------------------- |
| Идеальные условия                               | −1 до −2                   |
| Хорошие инструменты в наличии                   | −1 до −2                   |
| Неблагоприятные условия                         | +1 до +2                   |
| Плохие инструменты или их отсутствие            | +1 до +3                   |
| У противника есть соответствующая Специализация | +1                         |
| Разница в масштабе (за шаг)                     | ±1 к попаданию, ∓1 к урону |

### Установка сложности для драматического эффекта

Не все столкновения должны быть равномерно сбалансированы. Сложность не обязана быть одинаковой для всех. Королевский гвардеец с силовым копьём имеет преимущество перед контрабандистом с вибро-ножом. Мофф в укреплённой крепости знает местность лучше вторгающихся Повстанцев.

При установке сложности учитывайте:

1. **Местность и окружение** — есть ли у одной стороны преимущество домашней арены?
2. **Неравенство снаряжения** — военное vs. гражданское
3. **Подготовка** — засады vs. реагирование
4. **Боевой дух и мотивация** — отчаянный защитник сражается яростнее

## Балансировка столкновений

### Правило 80/20

Около 80% игрового процесса обрабатывается 20% правил. Большинству столкновений не нужны блоки статистики — достаточно числа сложности и описания. Полные характеристики оставьте для **босс-боёв** и **ключевых сцен**.

### Численное преимущество

В системе WoD численность имеет значение. Группа из 4–6 мобов может угрожать ПИ, потому что кости в конце концов будут на стороне атакующих. Используйте **правило 1** (шанс провала), чтобы позволить ПИ чувствовать себя героически, даже когда они в меньшинстве.

### Масштаб и сила

Используйте [Масштабы боя](../combat/combat-scales.mdx) при смешивании существ, транспорта и персонажей. Один шаг масштаба представляет огромную разницу в силе — ранкор (Масштаб 2) против персонажа (Масштаб 0) практически неуязвим для лёгкого оружия.

### Таблица уровней угрозы

| Столкновение    | Мобы | Именованные НИП | Сложность |
| --------------- | ---- | --------------- | --------- |
| **Лёгкое**      | 2–3  | 0               | 5–6       |
| **Стандартное** | 4–6  | 0–1             | 6         |
| **Сложное**     | 6–10 | 1–2             | 6–7       |
| **Трудное**     | 8–12 | 2–3             | 7–8       |
| **Босс**        | 0–4  | 1 (мощный)      | 8–9       |

Это свободные рекомендации. Пусть сюжет диктует столкновение, а не математика.

### Героическое правило сопротивления

ПИ и именованные НИП всегда теряют как минимум **1 уровень здоровья** от любой атаки, наносящей Летальный урон, независимо от сопротивления. Это предотвращает превращение любого персонажа в неуязвимого. Мобы не могут сопротивляться Летальному урону вообще.

## Социальные столкновения

Социальные столкновения следуют той же системе действий. Используйте **Харизму + соответствующую Способность** для убеждения, **Манипуляцию + Запугивание** для угроз, **Внешность + Выражение** для выступления.

| Цель                                           | Пул костей              | Сложность                                     |
| ---------------------------------------------- | ----------------------- | --------------------------------------------- |
| Убедить охранника пропустить вас               | Манипуляция + Хитрость  | 6 (охранник лоялен) до 8 (имперский гарнизон) |
| Торговаться о снижении цены                    | Харизма + Уличное чутьё | +1 за каждые 10% скидки                       |
| Запугиванием добыть информацию                 | Сила + Запугивание      | Сила Воли противника                          |
| Произвести впечатление на светском мероприятии | Внешность + Этикет      | 6 (повседневное) до 8 (имперский гала)        |

### Социальный бой

Для расширенных социальных конфликтов (дебаты, допросы, судебные драмы) используйте модель **Расширенного и Сопротивляемого** действия. Каждая сторона накапливает успехи. Первый, достигший Силы Воли противника, побеждает в споре.

---

## Столкновения с транспортом

Обратитесь к главе [Космический и транспортный бой](../vehicles-mechanisms/space-combat.mdx) для полных правил. Краткие рекомендации:

- **Незначительный транспорт** (неименованные TIE истребители, спидер-байки) не получает броски сопротивления
- **Разница масштабов** влияет на сложность попадания и урона на ±1 за шаг
- **Капитальные корабли** используют Хитроумие + Командование для определения действий за ход
- **Погони** используют дистанционные зоны — жертва сбегает, накопив достаточно успехов Манёвра

## Правило 80/20 на практике

Самый важный инструмент в вашем арсенале — знание того, когда **не** бросать кости. Если исход очевиден и неудача неинтересна, просто опишите это и двигайтесь дальше.

| Ситуация                                                                   | Бросок?                                  |
| -------------------------------------------------------------------------- | ---------------------------------------- |
| Опытный пилот, летящий через открытый ангарный отсек                       | Нет — автоматический успех               |
| Тот же пилот, уклоняющийся от обломков в разрушающейся космической станции | Да — сложность 7                         |
| ПИ пытается вспомнить базовую имперскую процедуру                          | Нет — они это знают                      |
| ПИ пытается вспомнить коды доступа к конкретному Звёздному Разрушителю     | Да — Интеллект + Бюрократия, сложность 8 |

## Осложнения и провалы

**Провал** (ноль успехов + хотя бы одна единица) никогда не должен означать просто «вы провалились». Он должен вводить **осложнение**, которое делает сцену более интересной.

### Боевые провалы

| Бросок        | Осложнение                                              |
| ------------- | ------------------------------------------------------- |
| Атака         | Осечка оружия, выроненное оружие, попадание по союзнику |
| Уклонение     | Спотыкание, падение ничком, потеря защиты               |
| Пилотирование | Риск столкновения, мерцание системы, потеря позиции     |

### Социальные провалы

| Бросок      | Осложнение                                |
| ----------- | ----------------------------------------- |
| Убеждение   | Оскорбить цель, раскрыть слишком много    |
| Запугивание | Цель раскрывает блеф, вызвана охрана      |
| Скрытность  | Опрокинуть что-то, сработала сигнализация |

### Провалы расширенных действий

При провале во время расширенного действия персонаж **теряет все накопленные успехи**. Рассказчик может решить, что:

- Задача не может быть повторена без новых инструментов или информации
- Осложнение ухудшает ситуацию (например, двигатель теперь имеет дополнительные повреждения)
- Время истекло (дверь открывается, прибывают имперцы)

## Работа в команде

Два или более персонажа могут работать вместе над одной задачей. Каждый совершает бросок отдельно и **добавляет свои успехи вместе**. Они никогда не объединяют пулы костей в один бросок.

Работа в команде хорошо подходит для:

- Подъёма тяжёлых предметов (Сила + Атлетика)
- Ремонта транспорта (Интеллект + Техника)
- Исследования темы (Интеллект + Расследование)
- Прикрывающего огня (стрелок получает бонусные кости от успехов прикрывающего персонажа)

Работа в команде мешает в некоторых контекстах. Несколько человек, пытающихся убедить одну и ту же цель, могут её запутать. Используйте здравый смысл.

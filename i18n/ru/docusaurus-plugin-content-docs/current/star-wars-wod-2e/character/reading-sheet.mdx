# Чтение листа

Каждый персонаж хранится в вашем браузере и отображается на странице [Универсальный лист](/universal_sheet). Каждый раздел сворачиваемый — нажмите на заголовок, чтобы развернуть или свернуть его, и ваше предпочтение сохранится до следующего раза.

---

## Карта листа

| #   | Раздел         | Ключевое содержание                                                                                |
| --- | -------------- | -------------------------------------------------------------------------------------------------- |
| 1   | **Base**       | Имя, раса, концепция, натура, поведение, портрет, внешность, биография                             |
| 2   | **Attributes** | Физические (3), Социальные (3), Ментальные (3) — рейтинг 1–5                                       |
| 3   | **Skills**     | Таланты (10), Навыки (10), Знания (10) + пользовательские на колонку — рейтинг 0–3                 |
| 4   | **Advantages** | Бэкграунды (пункты), Достоинства (очки), Недостатки (очки)                                         |
| 5   | **Force**      | Силовые навыки (5), Добродетели (3), Сила Воли, Очки Силы, Сопротивление Тёмной Стороне, Силы Силы |
| 6   | **Body**       | Инвентарь, Броня, Оружие, Импланты и кибернетика, Здоровье (7 уровней)                             |
| 7   | **Other**      | Производные характеристики (только чтение), Опыт (всего/потрачено/доступно), Заметки               |

Заголовок листа персонажа также содержит глобальные элементы управления: **Export**, **Import**, **Manage Characters**, **Reset** и **New Character**.

---

## Блок Base

Верхний раздел содержит всё, что идентифицирует вашего персонажа.

| Поле           | Описание                                                                                                                               |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **Name**       | Имя вашего персонажа.                                                                                                                  |
| **Concept**    | Краткое описание в одну строку — «Кореллианский контрабандист», «Падаван-джедай», «Бывший имперский офицер».                           |
| **Species**    | Выбрана из [каталога рас](./species.mdx). Определяет корректировки атрибутов и стартовые очки свободы.                                 |
| **Player**     | Ваше имя (человека за столом).                                                                                                         |
| **Nature**     | Глубинный архетип личности персонажа. Определяет, как вы восстанавливаете Очки Силы. См. [Этапы создания](./creation-steps/index.mdx). |
| **Demeanor**   | Лицо, которое ваш персонаж показывает миру — может отличаться от Натуры.                                                               |
| **Home World** | Планета происхождения вашего персонажа.                                                                                                |
| **Adventure**  | Название текущей истории или кампании.                                                                                                 |
| **Age**        | Возраст вашего персонажа в стандартных годах.                                                                                          |

### Портрет

Нажмите на область портрета, чтобы ввести URL изображения. Предварительный просмотр обновляется по мере ввода. Персонажи без портрета показывают placeholder-силуэт.

### Внешность

Опционально: рост, телосложение, волосы, глаза, пол и отличительные черты.

### Биография

Область свободного текста для предыстории, заметок о личности и всего остального, что не помещается в другие поля.

---

## Блок Attributes

Девять атрибутов, сгруппированных в три категории, рейтинг **1–5** (стартовый минимум 1 в каждом).

| Категория      | Атрибуты                        | Используется для                       |
| -------------- | ------------------------------- | -------------------------------------- |
| **Физические** | Сила, Ловкость, Выносливость    | Подъём, уклонение, поглощение урона    |
| **Социальные** | Харизма, Манипуляция, Внешность | Разговоры, обман, создание впечатления |
| **Ментальные** | Восприятие, Интеллект, Смекалка | Замечать, знать, реагировать           |

Каждый атрибут имеет:

- **Рейтинг** — нажмите на пункт, чтобы установить значение (1–5). Минимум всегда 1.
- **Специализация** — опциональная специальность (например, «Сила: Запугивание»).
- **Флаги** — `Specialization`, `Experienced`, `Practiced` — для отслеживания улучшений в игре.
- **Кнопка броска** — создаёт строку нотации кубов, используя значение атрибута. Нажмите, чтобы бросить.

Описания атрибутов и таблицы уровней см. в [Атрибуты и Способности](../core-rules/attributes-abilities.mdx).

## Блок Skills

Тридцать способностей, сгруппированных в три колонки, рейтинг **0–3** при создании (0 означает необученный).

| Колонка     | Способности                                                                                                              |
| ----------- | ------------------------------------------------------------------------------------------------------------------------ |
| **Таланты** | Бдительность, Атлетика, Рукопашный бой, Командование, Дипломатия, Уклонение, Эмпатия, Запугивание, Знание улиц, Хитрость |
| **Навыки**  | Бластер, Артиллерия, Рукопашное оружие, Пилотирование, Программирование, Ремонт, Езда, Взлом, Скрытность, Выживание      |
| **Знания**  | Астрогация, Бюрократия, Культуры, Интерфейсы, Расследование, Языки, Медицина, Политика, Технологии, Торговля             |

Каждая способность имеет:

- **Рейтинг** — нажмите на пункт, чтобы установить значение (0–3+). Минимума нет.
- **Текстовое поле специализации**.
- **Флаги** — как у атрибутов.
- **Кнопка броска** — строит нотацию `Атрибут + Способность`.

### Пользовательские способности

Внизу каждой колонки нажмите **Add Custom**, чтобы создать новую способность. Пользовательские способности работают идентично предустановленным. Используйте их для необычных навыков, не охваченных стандартными 30.

Полные описания способностей см. в [Атрибуты и Способности](../core-rules/attributes-abilities.mdx).

## Блок Advantages

Три колонки для долгосрочных ресурсов персонажа.

| Колонка         | Формат                       | Пример                                |
| --------------- | ---------------------------- | ------------------------------------- |
| **Backgrounds** | Рейтинг пунктов (0–5)        | Allies ••, Resources ••, Ship •       |
| **Merits**      | Стоимость в очках (положит.) | Iron Will (3 пт), Ambidextrous (2 пт) |
| **Flaws**       | Стоимость в очках (отрицат.) | Enemy (-3 пт), Phobia (-2 пт)         |

Нажмите **Add** в любой колонке, чтобы создать новую запись. Достоинства увеличивают доступные очки свободы; недостатки уменьшают стоимость других черт. См. [Достоинства и Недостатки](./merits-flaws.mdx) для полного каталога.

## Блок Force

Блок Силы разделён на три колонки.

### Левая колонка — Силовые навыки

Пять Силовых навыков рейтингом **0–5**: Control, Dynamism, Rapport, Sense, Telekinesis. Они работают как обычные способности, но доступны только чувствительным к Силе персонажам.

### Центральная колонка — Добродетели и запасы

**Добродетели** (рейтинг 1–5): Совесть, Страсть, Самообладание. См. [Добродетели и Сила Воли](./virtues-willpower.mdx).

**Отслеживание запасов:**

- **Сила Воли** (макс. = Страсть + Самообладание) — статическая защита от ментального вторжения и сил Силы.
- **Макс. Очки Силы** (мин. = Самообладание) — ваш максимальный запас.
- **Очки Силы** (текущие) — тратьте на автоматические успехи или бонусные кубы. Восстанавливается через отыгрыш Натуры.
- **Сопротивление Тёмной Стороне** (макс. = 5 + Совесть - Страсть) — сопротивление падению на тёмную сторону. Цветовая кодировка: синий (сильное), серый (среднее), красный (слабое).

### Правая колонка — Силы Силы

Прокручиваемый список известных Сил Силы. Отметьте силу, чтобы обозначить её как изученную. Фильтруйте по навыку или стоимости Очков Силы с помощью элементов управления вверху.

См. [Сила](./force.mdx) для полных описаний сил, правил активации и достоинств и недостатков, связанных с Силой.

## Блок Body

### Инвентарь

Список носимых предметов. Нажмите **Add Item**, чтобы добавить строку, введите описание и нажмите `×` для удаления. Используйте для кредитов, инструментов, датападов и прочего оборудования.

### Броня

Таблица с колонками: **Armor Type**, **Class**, **A.R.** (Armor Rating, вычитается из входящего урона), **Dex penalty** (снижает запасы кубов Ловкости). Нажмите **Add Armor**, чтобы добавить строку.

### Оружие

Таблица с колонками: **Weapon**, **Dmg** (нотация кубов урона), **Rng** (дальность), **Ammo** (выстрелов на магазин). Нажмите **Add Weapon**, чтобы добавить строку. См. [Снаряжение](../equipment.mdx) для каталога оружия и брони.

### Импланты и кибернетика

Таблица для кибернетических улучшений (живые персонажи) и встроенных инструментов/оружия (дроиды). Колонки:

| Колонка    | Описание                                                                                                |
| ---------- | ------------------------------------------------------------------------------------------------------- |
| **Name**   | Имплант или встроенный компонент (например, «Кибернетическая левая рука», «Встроенный сенсорный пакет») |
| **Type**   | Категория — Limb, Sensory, Uplink, Life Support, Weapon, Tool и т.д.                                    |
| **Effect** | Механическая выгода или описание того, что он делает                                                    |

Нажмите **Add Implant**, чтобы добавить строку. Используйте этот раздел для отслеживания протезов, нейронных имплантов, оборудования на дроидах и любых других постоянных или полупостоянных модификаций тела.

См. [Дроиды и киборги](./droids-cyborgs.mdx) для правил кибернетических улучшений и встроенного оборудования дроидов.

### Здоровье

7-уровневая полоса повреждений. Каждый уровень — это кликабельная кнопка, циклически переключающая три состояния:

| Состояние  | Значение                       |
| ---------- | ------------------------------ |
| Пусто (⬜) | Нет повреждений на этом уровне |
| Косая (/)  | Дробящий/оглушающий урон       |
| Крест (✕)  | Летальный урон                 |

| Уровень           | Штраф                    |
| ----------------- | ------------------------ |
| **Bruised**       | 0                        |
| **Hurt**          | −1                       |
| **Injured**       | −2                       |
| **Wounded**       | −3                       |
| **Mauled**        | −4                       |
| **Crippled**      | −5                       |
| **Incapacitated** | — (не может действовать) |

Штраф от первого отмеченного уровня применяется ко всем запасам кубов. См. [Здоровье, урон и лечение](../combat/health-damage-heal.mdx).

## Блок Other

### Производные характеристики

Они вычисляются автоматически и доступны только для чтения:

| Характеристика                   | Формула                           |
| -------------------------------- | --------------------------------- |
| **Сила Воли** (макс.)            | Страсть + Самообладание           |
| **Сопротивление Тёмной Стороне** | 5 + Совесть − Страсть             |
| **Инициатива**                   | Смекалка + Бдительность           |
| **Инициатива (свет. меч)**       | Смекалка + Бдительность + Control |
| **Дальность прыжка**             | × мин(Control, Telekinesis)       |
| **Скорость бега**                | × мин(Control, Telekinesis)       |

### Опыт

| Поле          | Описание                         |
| ------------- | -------------------------------- |
| **Total**     | Очки Опыта, заработанные в игре  |
| **Spent**     | Очки Опыта, которые вы потратили |
| **Available** | Всего − Потрачено (вычисляется)  |

Тратьте опыт на повышение атрибутов, способностей, добродетелей, силовых навыков и других черт. См. [Награды и развитие](../gm/rewards-advancement.mdx).

### Заметки

Область свободного текста для всего, что не помещается в другие поля — сюжетные линии, контакты НИП, заметки о кампании, список дел.

---

## Быстрые советы

- Все разделы запоминают своё состояние (свёрнуто/развёрнуто) между посещениями.
- **Кнопка броска** на любой черте открывает бросатель кубов с уже собранным правильным запасом.
- Регулярно экспортируйте персонажа как JSON-резервную копию. Используйте **Import** для восстановления.
- Используйте **Manage Characters** для переключения между сохранёнными персонажами или их удаления.

## Шаг 8: Силовые навыки и Силы (опционально)

Если ваш персонаж **чувствителен к Силе** (имеет Достоинство Force Sensitive или эквивалент), вы можете потратить Очки Свободы на покупку Силовых навыков и Сил.

- **Силовые навыки** стоят 3 Очка Свободы за пункт.
- **Силы Силы** стоят 2 Очка Свободы каждая (требуется хотя бы один пункт в соответствующем Силовом навыке).
- Все пять Силовых навыков начинаются с **0**.

| Силовой навык   | Управляет                                                                         |
| --------------- | --------------------------------------------------------------------------------- |
| **Control**     | Силы, воздействующие на ваше тело — лечение, игнорирование боли, сила             |
| **Dynamism**    | Силы, воздействующие на энергию — сдерживание бластерных выстрелов, молния Ситхов |
| **Rapport**     | Силы, воздействующие на разум — внушение, успокоение зверей, чтение мыслей        |
| **Sense**       | Силы восприятия — осознание, ясновидение, зондирование памяти                     |
| **Telekinesis** | Силы движения — перемещение объектов, сжатие, силовые броски                      |

См. [Сила](../force.mdx) для полного списка Сил Силы и правил.

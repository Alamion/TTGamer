## Шаг 7: Очки Свободы (15 очков)

У вас есть **15 Очков Свободы** для точной настройки персонажа. Их можно потратить на повышение существующих характеристик или покупку Достоинств (см. [Шаг 6](./step-6-merits-flaws) о Достоинствах и Недостатках).

### Таблица стоимости Очков Свободы

| Черта                            | Стоимость за пункт | Примечания                                                 |
| -------------------------------- | ------------------ | ---------------------------------------------------------- |
| **Атрибут**                      | 5 очков            | Повышает один Атрибут на 1 (макс. 5)                       |
| **Способность**                  | 2 очка             | Повышает одну Способность на 1 (макс. 3 при создании)      |
| **Добродетель**                  | 2 очка             | Повышает Совесть, Страсть или Самообладание на 1 (макс. 5) |
| **Силовой навык**                | 3 очка             | Требуется Достоинство Force Sensitive                      |
| **Сила Силы**                    | 2 очка за силу     | Требуется хотя бы 1 пункт в соответствующем Силовом навыке |
| **Бэкграунд**                    | 1 очко             | Повышает рейтинг бэкграунда на 1                           |
| **Достоинство**                  | 1 очко за пункт    | Покупайте достоинства по их указанной стоимости            |
| **Очки Силы**                    | 2 очка             | Повышает максимум Очков Силы на 1                          |
| **Сила Воли**                    | 2 очка             | Повышает максимум Силы Воли на 1                           |
| **Сопротивление Тёмной Стороне** | 1 очко             | Повышает Сопротивление Тёмной Стороне на 1                 |

### Стратегия трат

Достоинства стоят Очков Свободы, равных их указанному значению (Достоинство `+3` стоит 3 Очка Свободы). **Недостатки** дают дополнительные Очки Свободы, равные их указанному значению (Недостаток `-3` даёт вам ещё 3 Очка Свободы на траты). Чистая сумма Достоинств минус Недостатки не может превышать **7 очков** при создании персонажа.

См. [Достоинства и Недостатки](../merits-flaws.mdx) для полного каталога.

**Пример:** _Джакс_ тратит свои 15 Очков Свободы: повышает Ловкость с 4 до 5 (5 очков), повышает Бластер с 3 до 4, используя Достоинство Practiced (2 очка), повышает Ресурсы с 2 до 3 (1 очко) и берёт Достоинство **Daredevil** (+3, стоит 3 очка). Это 5 + 2 + 1 + 3 = 11 очков. Он берёт Недостаток **Soft-Hearted** (-1, даёт 1 очко) и **Enemy: Crimson Dawn** (-3, даёт 3 очка) для 4 дополнительных очков. Всего потрачено: 11 + 4 = 15.

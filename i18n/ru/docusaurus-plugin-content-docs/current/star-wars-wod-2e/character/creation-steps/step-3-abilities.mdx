## Шаг 3: Распределение способностей (13 / 9 / 5)

У вашего персонажа тридцать **Способностей**, разделённых на три категории:

| Категория   | Способности                                                                                                              |
| ----------- | ------------------------------------------------------------------------------------------------------------------------ |
| **Таланты** | Бдительность, Атлетика, Рукопашный бой, Командование, Дипломатия, Уклонение, Эмпатия, Запугивание, Знание улиц, Хитрость |
| **Навыки**  | Бластер, Артиллерия, Рукопашное оружие, Пилотирование, Программирование, Ремонт, Езда, Взлом, Скрытность, Выживание      |
| **Знания**  | Астрогация, Бюрократия, Культуры, Интерфейсы, Расследование, Языки, Медицина, Политика, Технологии, Торговля             |

Ранжируйте эти категории как **Основная** (13 пунктов), **Вторичная** (9 пунктов) и **Третичная** (5 пунктов). Распределите пункты внутри каждой категории.

- Способности начинаются с **0** — вы должны вложить пункты, чтобы получить какую-либо компетенцию.
- Максимум при создании персонажа — **3**.
- По десять каждой категории перечислены на листе; две дополнительные пустые строки на колонку предназначены для **Вторичных Способностей** (специализированных под-черт).

<br />

**Пример:** _Джакс_ ставит **Навыки** как Основные (13 пунктов), **Таланты** как Вторичные (9 пунктов) и **Знания** как Третичные (5 пунктов). Итоговое распределение: Бластер 3, Пилотирование 3, Ремонт 2, Взлом 2, Скрытность 2, Рукопашное оружие 1 | Бдительность 2, Уклонение 2, Знание улиц 2, Рукопашный бой 1, Хитрость 1, Запугивание 1 | Астрогация 2, Расследование 1, Культуры 1, Языки 1.

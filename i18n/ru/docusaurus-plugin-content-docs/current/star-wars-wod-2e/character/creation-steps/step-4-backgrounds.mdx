## Шаг 4: Бэкграунды (5 пунктов)

**Бэкграунды** представляют нарративные активы вашего персонажа — богатство, союзников, репутацию, снаряжение и статус. У вас есть **5 пунктов** для распределения между любой комбинацией бэкграундов.

| Бэкграунд            | Эффект                                                             |
| -------------------- | ------------------------------------------------------------------ |
| **Союзники**         | Люди, которые помогут вам без немедленной оплаты                   |
| **Контакты**         | Информаторы и источники информации по всей галактике               |
| **Кастомизация**     | Предмет снаряжения, улучшенный сверх заводских характеристик       |
| **Дроид**            | У вас есть НИП-дроид (степень зависит от пунктов)                  |
| **Влияние**          | Связи с организациями — правительство, гильдия или криминал        |
| **Наставник**        | Опытный учитель, который даёт советы                               |
| **Воинское звание**  | Официальное звание в военной организации (Армия или Флот)          |
| **Дворянский титул** | Титул и социальное положение от благородного происхождения         |
| **Репутация**        | Вы известны — в хорошем или плохом смысле — по всей галактике      |
| **Ресурсы**          | Доступный доход и материальное богатство                           |
| **Прислужники**      | Верные последователи, выполняющие ваши приказы                     |
| **Транспорт**        | Личный транспорт в вашем распоряжении (масштаб зависит от пунктов) |

См. [Бэкграунды](../backgrounds.mdx) для полных описаний и таблиц рангов.

**Персонажи-дроиды:** Могут использовать свой бэкграунд Ресурсы, чтобы встроить оборудование прямо в корпус при создании персонажа.

# Достоинства и недостатки

Достоинства и недостатки добавляют глубину и индивидуальность вашему персонажу. **Достоинства** — это выгодные черты — природные таланты, кибернетические улучшения или связи — которые стоят Очки Свободы, равные их указанному значению. **Недостатки** — это помехи — психологические слабости, физические изъяны или социальное бремя — которые дают дополнительные Очки Свободы, равные их указанному значению.

## Правила

- **Чистый лимит:** Сумма ваших Достоинств минус Недостатки не может превышать **7 очков** при создании персонажа.
- **Многократная покупка:** Многие Достоинства и Недостатки можно брать несколько раз. Если это возможно, в описании будет указано.
- **Одобрение Рассказчика:** Все Достоинства и Недостатки требуют одобрения Рассказчика. Некоторые могут не подходить для определённых кампаний.
- **Приобретение в игре:** Недостатки могут быть получены в процессе игры (из-за травмы, потрясения или сюжетных событий) без предоставления Очков Свободы. Достоинства можно купить за Очки Опыта после начала игры.

## Физиология рас

Категория **Alien Physiology** содержит черты, обычно характерные для конкретных рас — они представляют биологические адаптации, общие для целых рас. В детальном просмотре они отмечены как «Расовые — только не-люди».

Человеческий персонаж обычно не может взять достоинство или недостаток из Физиологии рас. Однако Рассказчик может разрешить исключения для персонажей с кибернетическими заменами, имитирующими эти черты, или для необычных происхождений.

Для шаблонов рас, включающих эти черты (с уже рассчитанными корректировками Очков Свободы), см. страницу [Расы](./species.mdx).

# Каталог транспорта

Блоки характеристик транспорта, конвертированные из исходных книг WEG. Каждая запись следует правилам конвертации из раздела [Характеристики и системы транспорта](./traits-systems.mdx).

Каталог охватывает наземный транспорт, шагоходы, спидеры, звездолёты, космические транспорты и капитальные корабли из основной книги правил WEG.

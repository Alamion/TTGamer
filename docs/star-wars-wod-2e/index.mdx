# Star Wars WoD 2e

A hybrid tabletop RPG conversion that runs the Star Wars galaxy on the **Vampire: The Masquerade 2nd Edition** engine.

---

## About the Original Conversion

**Mark L. Shute** created this conversion back in the 1990s. His group preferred the elegance and flexibility of the World of Darkness system over other games they played — but they also loved Star Wars. The West End Games _Star Wars Roleplaying Game_ used a similar dice pool mechanic, but its skill system felt too broad and detailed for the cinematic feel they wanted.

Over two decades of tweaking, the conversion grew into a complete system that replicates everything seen on screen — from blaster firefights to lightsaber duels to starfighter combat. The result is a game where:

- **Blasters** feel deadly but heroes stay heroic (PCs can soak Lethal damage)
- **The Force** works like a Vampire Discipline system with hierarchical powers
- **Dark Side** temptation mirrors Humanity mechanics
- **Merits & Flaws** from the Vampire Players Guide differentiate aliens, cyborgs, and Force-sensitives
- **Combat scales** let an X-wing strafe a Star Destroyer and a Rancor swat at Ewoks with a single unified rule

### Required Books

To use the original conversion you need:

- **The Star Wars Roleplaying Game, Second Edition, Revised and Expanded** (WEG, ISBN 0-87431-268-X)
- **Vampire: The Masquerade, Second Edition** (White Wolf, ISBN 1-56504-029-)
- **The Vampire Players Guide, Second Edition** (White Wolf, ISBN 1-56504-053-)

## About This Web Version

This site is a digital companion to the conversion — not a replacement for the books. It provides:

| Feature                                           | Description                                                                                                  |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| **Character Sheet Manager** at `/universal_sheet` | Full interactive character sheet with auto-calculated derived stats, health tracking, and JSON import/export |
| **Dice Roller**                                   | Built-in d10 pool roller with difficulty, Rule of One, exploding 10s, and 3D visualization                   |
| **Documentation**                                 | Rules reference, creation steps, combat flow, equipment, and GM tools — all searchable                       |

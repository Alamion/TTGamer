Hi to everyone. So basically I wanted to create a really cool docs on TTRPGs I am playing and GMing, and I've done this.

I'll update the manager and docs from time to time, trying to make it a universal and handy solution for both experienced GMs and newbies.

If you have any suggestions or wish to participate in docs filling up, you are welcomed on [issues](https://github.com/Alamion/TTGamer/issues) page (for suggestions) and [PR](https://github.com/Alamion/TTGamer/pulls) page (for redacting docs by yourself).

That's all.
